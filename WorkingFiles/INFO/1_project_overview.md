# Project Overview: Hospital Maps

## Goal

In this project, you will analyze health data in Germany. The result of the project should be a data ingestion pipeline resulting in a dashboard displaying maps and/or diagrams.

## Research Questions

Here is a list of questions the project might answer. Please focus on a few of these:

- which types of diagnosis are the most frequent?
- where are the most cardiac surgeries carried out? (or other types of diagnosis/procedure)
- where are hospitals located?
- calculate the geographical density of hospitals
- which area has the most hospitals/beds/diagnoses per population?
- calculate the travel distance from a given location to Germany
- compare the 2023 data to 2024 and visualize the difference
- use protomaps to create a data privacy sensitive mapping service
- plug in an AI system to talk to the data and ask questions like the above

## The Data

There are three main data sources for the project:

- diagnoses in the ICD system
- medical procedures in the OPS system
- hospital data

Optionally, you could include additional data sources like:

- geo coordinates of hospitals (generated from addresses)
- demographic data about population density
- postal code areas as GeoJSON
- points of interest from Open Street Map

## How to work on on the project?

- I recommend working in a small team (2-3 persons)
- I recommend strongly to share and exchange ideas, solutions and code
- if you decide to work on your own, I will ask for mandatory peer reviews (you review each others design and/or code)
- plan and work through the project tasks on your own
- when you reach an important milestone, e.g. the data ingestion make an appointment to discuss further steps
- when you get stuck, reach out as well
